🔒 Biện Pháp Bảo Mật
CRITICAL

Path Policy Enforcement

Chặn ghi vào /etc, /System, /bin. Mutating actions chỉ trong allowed_roots. SAFE mode khóa mọi thứ.

CRITICAL

Shell Command Safety

Chặn rm -rf /, sudo, reboot, fork bomb. Chặn pipe-to-shell (curl|bash). Chặn null bytes, control chars.

HIGH

API Input Validation

Pydantic Field constraints: max_length, ge/le bounds, pattern regex. Chặn payload quá lớn.

HIGH

Rate Limiting

Sliding-window per-client. 120 RPM default. Auto-prune stale buckets. 429 response với retry-after.

MEDIUM

Path Traversal Protection

Backup restore chặn ../ traversal. resolve() paths. Validate filenames trước khi extract.

MEDIUM

Loop Command Sanitization

Chặn shell metacharacters (;|&<>`$()), unterminated quotes, trailing escape. shlex.split safe parsing.
