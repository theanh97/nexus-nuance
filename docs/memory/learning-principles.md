## SCIENTIFIC LEARNING METHODS

### 1. Spaced Repetition (Lặp lại ngắt quãng)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Review knowledge at OPTIMAL INTERVALS:                             │
│                                                                     │
│  • First review:   after 1 day                                      │
│  • Second review:  after 3 days                                     │
│  • Third review:   after 7 days                                     │
│  • Fourth review:  after 14 days                                    │
│  • Fifth review:   after 30 days                                    │
│                                                                     │
│  Formula: interval = previous_interval * 2.5                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2. Active Recall (Gợi nhớ chủ động)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Don't just re-read. TEST yourself:                                 │
│                                                                     │
│  • Ask: "What did I learn about X?"                                 │
│  • Try to recall WITHOUT looking at notes                           │
│  • Then check accuracy                                              │
│  • This STRENGTHENS neural pathways                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 3. Chunking (Chia nhỏ)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Break complex info into DIGESTIBLE CHUNKS:                         │
│                                                                     │
│  • Max 3-5 key points per chunk                                     │
│  • Each chunk should be COMPLETE in itself                         │
│  • Build up from simple to complex                                  │
│  • Use hierarchical organization                                    │
│                                                                     │
│  Human working memory: 7 ± 2 items                                  │
│  Optimal chunk size: 3-5 items                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 4. Interleaving (Xen kẽ)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Mix RELATED concepts while learning:                               │
│                                                                     │
│  • Don't study AAAAA then BBBBB                                     │
│  • Instead: ABABC BCACB ABCBA                                       │
│  • Forces brain to DISCRIMINATE between concepts                    │
│  • Improves long-term retention                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 5. Elaboration (Diễn giải sâu)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Connect NEW knowledge to EXISTING knowledge:                       │
│                                                                     │
│  • Ask: "How does this relate to what I already know?"              │
│  • Ask: "Why is this true?"                                         │
│  • Ask: "What if...?"                                               │
│  • Create mental models and analogies                               │
│  • Teach it to someone else (or to yourself)                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 6. Metacognition (Siêu nhận thức)
```
┌─────────────────────────────────────────────────────────────────────┐
│  SELF-AWARENESS about learning:                                     │
│                                                                     │
│  • "Do I really understand this?"                                   │
│  • "What's my confidence level?"                                    │
│  • "What am I still confused about?"                                │
│  • "How can I learn this better?"                                   │
│                                                                     │
│  Track your own learning process                                    │
│  Identify weak areas and focus on them                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 7. Deliberate Practice (Luyện tập có chủ đích)
```
┌─────────────────────────────────────────────────────────────────────┐
│  FOCUSED practice on WEAKNESSES:                                    │
│                                                                     │
│  • Identify specific skill gaps                                     │
│  • Practice at the edge of ability (not too easy, not too hard)    │
│  • Get immediate feedback                                           │
│  • Repeat with adjustments                                          │
│  • Track progress over time                                         │
│                                                                     │
│  NOT just repetition - PURPOSEFUL improvement                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 8. Dual Coding (Mã hóa kép)
```
┌─────────────────────────────────────────────────────────────────────┐
│  Combine VERBAL and VISUAL:                                         │
│                                                                     │
│  • Text + diagrams                                                  │
│  • Code + flowcharts                                                │
│  • Concepts + mind maps                                             │
│  • Notes + sketches                                                 │
│                                                                     │
│  Two memory traces = stronger retention                             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## THE LEARNING SYSTEM IMPLEMENTATION

### Daily Cycle (Hàng ngày)
```python
DAILY_LEARNING_CYCLE = {
    "hour_0": {
        "action": "COLLECT",
        "tasks": [
            "Scan news sources",
            "Check forums (Hacker News, Reddit)",
            "Read blog posts",
            "Check GitHub trending",
            "Review research papers"
        ],
        "output": "raw_knowledge_items"
    },

    "hour_1": {
        "action": "DIGEST",
        "tasks": [
            "Summarize each item",
            "Extract key points",
            "Identify relevance to current goals",
            "Rate quality and importance"
        ],
        "output": "digested_knowledge"
    },

    "hour_2": {
        "action": "FILTER",
        "tasks": [
            "Score each item (0-10)",
            "Keep only score >= 6",
            "Categorize by agent/topic",
            "Generate improvement proposals"
        ],
        "output": "filtered_knowledge"
    },

    "hour_3": {
        "action": "APPLY",
        "tasks": [
            "Review proposals",
            "Get user approval if needed",
            "Implement improvements",
            "Test changes",
            "Document lessons learned"
        ],
        "output": "applied_improvements"
    }
}
```
